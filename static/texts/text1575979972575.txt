<h1>【MySQL 5.7】修改密码报错解决</h1><p></p><p style="text-align:start;" size="1" _root="[object Object]" __ownerID="undefined" __hash="undefined" __altered="false">报错：ERROR 1819 (HY000): Your password does not satisfy the current policy requirements</p><p></p><p style="text-align:start;" size="0" _root="undefined" __ownerID="undefined" __hash="undefined" __altered="false">解决办法：</p><p style="text-align:start;" size="0" _root="undefined" __ownerID="undefined" __hash="undefined" __altered="false">1、查看 mysql 初始的密码策略，<br/>输入语句 “ SHOW VARIABLES LIKE &#x27;validate_password%&#x27;; ” 进行查看，</p><pre size="0" _root="undefined" __ownerid="undefined" __hash="undefined" __altered="false"><code>SHOW VARIABLES LIKE &#x27;validate_password%&#x27;;</code></pre><p></p><div class="media-wrap image-wrap"><img src="http://localhost:3001/static/images/image.1575978612327.png"/></div><p></p><p>2、首先需要设置密码的验证强度等级，设置 validate_password_policy 的全局参数为 LOW 即可，</p><p>输入设值语句 “ set global validate_password_policy=LOW; ” 进行设值，</p><pre><code>set global validate_password_policy=LOW;</code></pre><p></p><div class="media-wrap image-wrap"><img src="http://localhost:3001/static/images/image.1575978715816.png"/></div><p></p><p>3、当前密码长度为 8 ，如果不介意的话就不用修改了，按照通用的来讲，设置为 6 位的密码，设置 validate_password_length 的全局参数为 6 即可，</p><p>输入设值语句 “ set global validate_password_length=6; ” 进行设值，</p><pre><code>set global validate_password_length=6;</code></pre><p></p><div class="media-wrap image-wrap"><img src="http://localhost:3001/static/images/image.1575978748982.png"/></div><p></p><p>4、现在可以为 mysql 设置简单密码了，只要满足六位的长度即可，</p><p>输入修改语句 “ ALTER USER &#x27;root&#x27;@&#x27;localhost&#x27; IDENTIFIED BY &#x27;123456&#x27;; ” 可以看到修改成功，表示密码策略修改成功了！！！</p><p></p><div class="media-wrap image-wrap"><img src="http://localhost:3001/static/images/image.1575979945789.png"/></div><p></p><p>关于 mysql 密码策略相关参数；</p><p>1）、validate_password_length  固定密码的总长度；</p><p>2）、validate_password_dictionary_file 指定密码验证的文件路径；</p><p>3）、validate_password_mixed_case_count  整个密码中至少要包含大/小写字母的总个数；</p><p>4）、validate_password_number_count  整个密码中至少要包含阿拉伯数字的个数；</p><p>5）、validate_password_policy 指定密码的强度验证等级，默认为 MEDIUM；</p><p>关于 validate_password_policy 的取值：</p><p>0/LOW：只验证长度；</p><p>1/MEDIUM：验证长度、数字、大小写、特殊字符；</p><p>2/STRONG：验证长度、数字、大小写、特殊字符、字典文件；</p><p>6）、validate_password_special_char_count 整个密码中至少要包含特殊字符的个数；</p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p>
