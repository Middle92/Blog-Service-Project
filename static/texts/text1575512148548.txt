<h1>navicat 远程链接 Linux 的 MySQL 8.0</h1><p></p><div class="media-wrap image-wrap"><img src="http://localhost:3001/static/images/image.1575511122696.png"/></div><p>navicat官网：<a href="https://www.navicat.com.cn/products/">https://www.navicat.com.cn/products/</a></p><p></p><p><span style="color:#c0392b"><strong>Linux配置MySQL</strong></span></p><p>编辑/etc/my.cnf</p><p>添加skip-grant-tables</p><p>保存</p><pre><code>vi /etc/my.cnf</code></pre><p></p><div class="media-wrap image-wrap"><img src="http://localhost:3001/static/images/image.1575511410691.png"/></div><p></p><p>重启MySQL</p><pre><code>service mysql restart</code></pre><p>进入MySQL</p><pre><code>mysql -u root</code></pre><p>切换mysql数据库</p><pre><code>use mysql</code></pre><p>清空密码</p><pre><code>update user set anthentication_string=&#x27;&#x27; where  user=&#x27;root&#x27;</code></pre><p>退出MySQL</p><pre><code>exit</code></pre><p>然后编辑my.cnf，把刚刚添加的skip-grant-tables删除或者注释</p><p></p><div class="media-wrap image-wrap"><img src="http://localhost:3001/static/images/image.1575511639743.png"/></div><p></p><p><span style="color:#c0392b"><strong>重置MySQL密码</strong></span></p><p>每次修改my.cnf文件都需要重启MySQL</p><pre><code>service mysql restart</code></pre><p>进入MySQL</p><pre><code>mysql -u root</code></pre><p>切换数据库</p><pre><code>use mysql;</code></pre><p><span style="color:#c0392b"><strong>设置密码</strong></span></p><p>更改root用户的host为%</p><pre><code>alter user &#x27;root&#x27;@&#x27;localhost&#x27; identified by &#x27;新密码&#x27; password expire never;<br/>alter user &#x27;root&#x27;@&#x27;localhost&#x27; with mysql_native_password by &#x27;新密码&#x27;;<br/><br/>update user set host=&quot;%&quot; where user=&#x27;root&#x27;</code></pre><p>执行一下MySQL</p><pre><code>flush privileges;</code></pre><p>Linux配置完成。</p><p></p><p>配置navicat</p><p></p><div class="media-wrap image-wrap"><img src="http://localhost:3001/static/images/image.1575512105383.png"/></div><p></p><p>点击下链接测试</p><p></p><p></p><div class="media-wrap image-wrap"><img src="http://localhost:3001/static/images/image.1575512133854.png"/></div><p></p><p>连接成功</p><p></p><p></p><p></p><p></p><p></p>
